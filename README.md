# E-Commerce Retail Analytics Dashboard

## Project Overview
An end-to-end sales analytics dashboard 
built in Microsoft Excel for a California-
based fashion and apparel e-commerce business.
Analyzes 2,400+ transactions across 13 weeks.

## Business Questions Answered
- Which products generate highest revenue?
- Which sales channel performs best?
- How do male vs female customers buy differently?
- Which counties have highest customer concentration?
- How long does delivery take on average?
- How satisfied are customers overall?

## Key Metrics Tracked
- Total Orders: 2,400
- Total Quantity Sold: 11,997
- Total Revenue: $649,000
- Average Customer Rating: 4.0 / 5.0
- Average Days to Deliver: 2.3 days

## Dashboard Components
- 13-week revenue and quantity trend chart
- Gender vs sales channel heatmap matrix
- Product popularity breakdown by gender
- Geographic customer distribution map
- Shipping time distribution analysis
- Customer satisfaction trend chart
- 5 dynamic KPI cards

## Dataset Details
- 2,400+ transactions (TX500001 onwards)
- 20+ product categories
  (Jeans, Sneakers, T-Shirts, Dresses etc.)
- 5 order channels
  (App, Website, Instagram, 
   Partner App, Target.com)
- Customer segments
  (Female 52%, Male 39%, 
   Other 3%, Unknown 6%)
- California county-level geography

## Tools and Skills Used
- Microsoft Excel
- Pivot Tables and Pivot Charts
- Power Query for data transformation
- Conditional formatting (heatmap)
- Bing Maps integration
- Dynamic KPI cards using shapes
- Advanced charting and visualization

## Key Insights
1. Website and App are top revenue channels
2. Female customers drive 52% of total sales
3. T-Shirts and Sneakers are highest 
   volume products
4. Average delivery time is 2.3 days
5. Customer satisfaction averages 4.0/5.0
